REAL-TIME MILK ADULTERATION DETECTOR

PROJECT OVERVIEW
----------------------------------------------------------------------------------------------------------------------

Milk adulteration is a major issue in the dairy industry where harmful substances such as water, starch, detergent, urea and salt are added to milk to increase quantity and profit. Conventional testing methods require laboratory equipment, skilled technicians and significant processing time.
This project presents a portable IoT-based milk adulteration detection system using:
-AS7262 Multispectral Sensor
-EC Sensor
-ESP32 Microcontroller
-Machine Learning (Logistic Regression)
-Mobile Application
The system analyzes the optical and electrical properties of milk to identify adulteration in real time. Sensor data is processed using machine learning and results are displayed on a mobile application through Bluetooth or Wi-Fi communication.The proposed system is designed to be:
Low-cost
Portable
Real-time
User-friendly making it suitable for dairy farms, milk collection centers and rural applications.

ABSTRACT
----------------------------------------------------------------------------------------------------------------------

Milk adulteration directly affects food safety and public health. Existing laboratory-based testing methods are expensive, time-consuming and not suitable for real-time field applications.This project proposes a real-time milk adulteration detection system using multispectral sensing, electrical conductivity sensing, IoT communication and machine learning. The AS7262 multispectral sensor measures optical characteristics of milk at multiple wavelengths, while the EC sensor measures conductivity changes caused by adulterants.
The collected sensor data is processed by an ESP32 microcontroller and classified using a Logistic Regression machine learning model. The final result is transmitted to a mobile application through Bluetooth or Wi-Fi for instant monitoring.
The proposed system provides a portable, low-cost and intelligent solution for milk quality analysis and adulteration detection.

OBJECTIVE
